# Physics Interaction Sandbox

## Overview

A Unity-based physics interaction prototype where the player controls a cube, collects targets, and completes the level.

## Features

* WASD movement
* Jump mechanics
* Rigidbody physics
* Collision detection
* Target collection system
* Score tracking
* Camera follow system
* Win condition
* Restart functionality

## Technologies Used

* Unity
* C#
* TextMeshPro
* Unity Physics System

## What I Learned

* Rigidbody movement
* Collision handling
* UI systems
* Scene management
* Basic gameplay loop design

## Controls

* W/A/S/D - Move
* Space - Jump
